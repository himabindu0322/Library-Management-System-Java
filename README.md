# 📚 Library Management System (Java)
## 📌 Description
A simple Library Management System developed using Java. This project helps to manage books, users, and issue/return operations. It includes basic CRUD functionalities and is created as a college mini project.

---

## 🚀 Features

* Add new books
* View available books
* Issue books to users
* Return books
* Manage user details
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Java
* JDBC (Database Connectivity)
* MySQL
